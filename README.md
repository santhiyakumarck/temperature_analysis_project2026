# District-wise Temperature Analysis and Prediction
District-wise Temperature Analysis and Prediction using Python, Power BI and Machine Learning


## Project Overview

This project analyzes historical temperature data from multiple Tamil Nadu districts using Python, Power BI, and Machine Learning.

The objective is to identify the hottest and coolest districts, analyze weather trends, and predict future temperatures using machine learning models.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Power BI
* Microsoft Excel

## Dataset Features

* Date
* District
* Maximum Temperature (°C)
* Minimum Temperature (°C)
* Wind Speed (km/h)

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Power BI Dashboard Creation
5. Machine Learning Model Development
6. Temperature Prediction

## Key Insights

* Identified district-wise temperature variations.
* Determined hottest and coolest districts.
* Analyzed wind speed patterns.
* Compared temperature trends across multiple districts.
* Predicted future temperatures using machine learning.

## Machine Learning Models

* Linear Regression
* Random Forest Regressor
